Senamhi Script
==============
A collection of functions to obtain Peruvian climate data in R.
More details forthcoming.

To use
------
install.packages("devtools")
library(devtools)
install_github(repo = "senamhi", username = "ConorIA")
library(senamhi)

Included functions
------------------
senamhi()
senamhiDownload()
senamhiWriteCSV()
